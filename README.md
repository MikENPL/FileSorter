# FileSorter
This program can sort many filetypes from many sources into the right folders.

___
## Prerequisites
- wxPython: `pip install wxPython`

## How to use
Add a sorting category in the **Categories** tab, choose a path to which the files will be put, then add the filetypes that will be put there. 

In the **Source** tab add all folders from which the files will be sorted.

Click the **Sort** button to begin sorting.
